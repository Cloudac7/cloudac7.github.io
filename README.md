# Cloudac7 Blog

[Astro](https://astro.build) + [Svelte](https://svelte.dev/) blog, deployed on [Cloudflare Workers](https://workers.cloudflare.com/).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Build output:

- `dist/` — Static assets (HTML, JS, CSS, search index) and `_worker.js/` (SSR runtime)
- `dist/pagefind/` — Search index (auto-generated by Pagefind)

## Deploy

```bash
npx wrangler deploy
```

The `wrangler.jsonc` configures the Worker entry and static asset serving.
